SOAP протокол. 
соап работает с протоколами прикладного уровня (htp,htpp,htpps)
соап для передачи инфо используется XML формат
WSDL - язык который описывает заголовки, собития и сообщения для веб сервиса. ОПИСАНИЕ СТРУКТУРЫ SOAP протокола. Он обязателен для SOAP 
WSDL - в нем есть элементы данных, есть тип данных, список операций
если есть SOAP значит есть WSDL !!! 
правило записки XML (расширенный язык разметки) формата. 
 - обьявление XML идет первой отдельно строкой и включает в себя информацию о себе
 - состоит из элементов, 
 - теги и контент и атрибуты.
 - имеет только один корневой элемент
 - все элементы имеют закрывающие теги
 - название тегов регистрозависимы
 - элементы не пересекаются
 - все "атрибуты" только в кавычках
